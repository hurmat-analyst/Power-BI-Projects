# Data Professional Survey Dashboard 

This project analyzes survey data from data professionals using Power BI.

The data was cleaned and transformed using Power Query, and an interactive dashboard was created to explore insights about salary, 
programming languages, work-life balance, and career entry difficulty in this data indutry.

**Key Insights:**
- Average salary by job title
- Most popular programming languages among data professionals
- Difficulty level to break into data field
- Work-life balance and salary satisfaction
- Demographic insights such as age and location of survey respondents

**Tools Used:**
- Power BI
- Power Query
- Data Visualizations
